Full Moon Theme for Omarchy
A  light inspired theme for DHH's Omarchy, an opinioned Arch Linux setup.

Preview

![](/home/pseudo/Pictures/screenshot-2025-12-26_20-29-37.png)

Installation
Install on an existing omarchy setup using omarchy-theme-install https://github.com/drunk-particles/omarchy-dotfiles.git

Usage
Once installed, press Ctrl+Shift+Super+Space, select "Full Moon" and everything should refresh automatically.
